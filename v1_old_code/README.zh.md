## 归档信息

此目录包含旧版 v1 代码库（基于 WPF），仅用于归档目的。

### 原始功能（v1）
- EDL 模式检测与连接
- Firehose 协议支持
- 基础分区表读写
- XML 配置文件处理
- 分区清理功能
- 多语言支持（中英文）
- 图形化界面（WPF）

### 系统要求（原始）
- Windows 7/8/10/11（推荐 64 位）
- .NET 8.0 运行时
- 管理员权限
- USB 2.0/3.0 接口

## 迁移到 v2

全新的 **v2** 版本包含以下改进：
- **全新 UI 框架**：从 WPF 迁移到 Avalonia UI
- **更好的架构**：更清晰的代码结构，更易于维护
- **性能提升**：分区解析和刷入操作更快
- **原生 AOT 编译**：单文件发布，无需安装 .NET 运行时
- **增强功能**：OFP/OPS 解密、Super分区支持等
